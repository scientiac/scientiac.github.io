Colophon

I built this site to be readable and customizable while being minimal. It has become more and more complex (relatively) over the years under the hood though. I use the LaTeX Fonts as the preferred Fonts and use Readable CSS as the base to my styles. I have used zola as the static site generator and github actions do the heavy lifting of building the site, then Sending, and Receiving webmentions. This site also complies with the IndieWeb guidelines.
